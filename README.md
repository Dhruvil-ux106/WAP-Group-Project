# 🎵 Spotify Clone (React)

A frontend-focused clone of Spotify built using React.
This project replicates the core UI and basic interactions of the Spotify web app, including playlists, navigation, and a music player interface.

---

## 🚀 Features

* 🎧 Sidebar navigation (Home, Search, Library)
* 🏠 Home page with playlists/albums
* 📄 Playlist page with song list
* ▶️ Music player (UI + basic state)
* 🔍 Search page (UI-based filtering)
* 🌙 Dark theme UI (Spotify-inspired)
* ⚡ Fully responsive layout (basic)

---

## 🛠️ Tech Stack

* React (Vite)
* React Router DOM
* JavaScript (ES6+)
* CSS (custom styling)

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/spotify-clone.git
```

2. Navigate into the project

```
cd spotify-clone
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```
